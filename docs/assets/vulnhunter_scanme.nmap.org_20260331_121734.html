Executive Summary

Security assessment of scanme.nmap.org and nmap.org infrastructure revealed multiple critical vulnerabilities including password reset token theft, server-side request forgery allowing internal network access, and widespread open redirects. The absence of HTTPS on scanme.nmap.org and unencrypted FTP services further expose the infrastructure to data interception and credential theft.
